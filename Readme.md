production-inspired backend clone of YouTube built to learn and practice backend engineering at scale.

This project focuses on building the core architecture behind a modern video platform — including authentication, video uploads, streaming, channels, subscriptions, comments, likes, playlists, and more.

Goal

The goal of this project is to understand how large-scale backend systems are designed and built in real-world production environments by recreating the backend of a video-sharing platform from scratch.

Tech Stack
Node.js
Express.js
MongoDB
Mongoose
JWT
Cloudinary
Features
User Authentication
Channel Management
Video Upload & Streaming
Comments & Replies
Likes & Dislikes
Subscriptions
Playlists
Watch History
Search & Filtering
Scalable backend architecture
RESTful API design
Purpose

This project is built for learning, experimenting, and understanding backend development at a production level through hands-on implementation.
